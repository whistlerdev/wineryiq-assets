# wineryIQ assets

Thumbnailed winery photos used by the wineryIQ iOS app, served via jsDelivr:

    https://cdn.jsdelivr.net/gh/whistlerdev/wineryiq-assets@main/img/<file>.jpg

Each image is sourced from the winery's own website (og:image), downscaled to
1200px on the long edge. Filenames `wNNNNN.jpg` map to the wineryIQ database id.
